# Tradesman Finder Website (PHP & MySQL)

A dynamic web-based platform designed to connect users with local tradesmen and service providers.
This system allows customers to search for artisans, register accounts, and manage service interactions through a structured database-driven web application.

The project demonstrates full-stack web development using PHP, MySQL, HTML, CSS and JavaScript.

---

## Key Features

* User registration and login system
* Tradesman registration and profile creation
* Search and browse tradesmen by service/category
* Admin and user dashboard functionality
* Password recovery and account management
* Database-driven content and authentication
* Dynamic PHP pages with form handling
* Secure data handling using server-side scripting

---

## Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL
* **Server Environment:** XAMPP / Localhost

---

## How to Run the Project (Local Setup)

### Step 1: Install XAMPP

Download and install XAMPP from:
https://www.apachefriends.org/

### Step 2: Move Project Folder

Move this project folder to:

```
C:\xampp\htdocs\
```

Example:

```
C:\xampp\htdocs\Tradesman-Finder-Website
```

### Step 3: Start Server

Open XAMPP Control Panel and start:

* Apache
* MySQL

### Step 4: Import Database

1. Go to browser: http://localhost/phpmyadmin
2. Create a new database (e.g., `tradesman_db`)
3. Import the SQL file included in this project (if available)

### Step 5: Run Project

Open browser and go to:

```
http://localhost/Tradesman-Finder-Website
```
